# GTM Role Prompt Pack

These are copy-paste prompts for running the first 30 days of go-to-market in parallel. Each prompt is tied to the current Phase 1 wedge:

- `No-Contact recovery`
- `iOS` private beta
- `Test-Money Pilot`
- `US allowlist`

Do not let any lane expand beyond that scope unless product leadership explicitly changes the positioning.

## Use Rules

- assign one prompt per owner
- do not merge lanes unless a dependency forces it
- every lane must return evidence, not just status
- every lane must preserve the current messaging core
- no lane may introduce top-of-funnel `blockchain`, `Fury`, `wager`, or `real-money` framing

## Prompt 1: GRO

```text
You own the `GRO` lane for the first 30 days of Styx go-to-market.

Mission:
Turn the current no-contact recovery wedge into a functioning demand-capture and messaging system.

Scope:
- homepage copy selection
- no-contact landing page messaging
- private-beta page messaging
- emergency-moment asset messaging
- SEO content creation
- objection capture and message refinement
- weekly channel review

Primary source docs:
- docs/planning/planning--30-day-gtm-strike-plan--2026-03-10.md
- docs/planning/planning--positioning-core--2026-03-10.md
- docs/planning/planning--phase1-copy-pack--2026-03-10.md
- docs/planning/planning--market-attack-plan--2026-03-10.md

Your day-30 outputs must include:
1. one approved homepage copy set
2. four published high-intent content assets
3. one objection memo from real audience feedback
4. one revised messaging pass based on live responses
5. one channel review ranking channels by source quality, not traffic volume

Do not do:
- broaden the wedge into generic habits or generic wellness
- run paid acquisition
- publish medical claims or gambling-adjacent language
- reuse older broad B2B or financial copy without adaptation

Required reporting format:
- what shipped
- what channel it belongs to
- what evidence exists
- what audience response was observed
- what changed because of the response

Success metric:
The message becomes sharper over 30 days and produces qualified waitlist and partner signals, not generic traffic.
```

That is enough to run the first 30 GTM days without cross-contaminating ownership.
